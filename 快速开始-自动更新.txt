========================================
如何使用自动更新功能
========================================

【用户端】
1. 启动程序 → 自动检查更新（3秒后）
2. 或点击 "🔄 检查更新" 按钮手动检查
3. 发现新版本 → 显示更新提示
4. 点击确认 → 自动下载并安装
5. 程序自动重启到新版本

【开发端 - 发布新版本】

方式1：使用发布助手（推荐）
--------------------------
1. 双击运行：发布新版本.bat
2. 输入新版本号（如：1.1.0）
3. 输入更新说明
4. 等待自动打包完成
5. 测试 dist\pyQuickStart.exe
6. 在 Gitee 创建 Release
7. 上传 exe 文件

方式2：手动发布
--------------------------
1. 编辑 version.json，更新版本号
2. 运行：打包程序.bat
3. 测试 dist\pyQuickStart.exe
4. 在 Gitee 创建 Release
5. 上传 exe 文件
6. 提交代码

========================================
Gitee Release 创建步骤
========================================

1. 访问：https://gitee.com/sytao_2020/pyQuickStart/releases

2. 点击 "创建新的发行版"

3. 填写信息：
   标签名称：v1.1.0
   发行版标题：v1.1.0 - 添加自动更新功能
   发行说明：
   ---
   ## 新功能
   - ✨ 新增自动更新系统
   - 🔄 程序启动时自动检查更新
   - 📥 支持一键下载安装
   
   ## 使用说明
   - 程序启动后会自动检查更新
   - 也可点击 "🔄 检查更新" 按钮
   - 发现新版本时会提示更新内容
   - 确认后自动下载并安装
   ---

4. 上传文件：拖拽 dist\pyQuickStart.exe

5. 点击 "发布"

========================================
版本号规则
========================================

格式：主版本.次版本.修订号 (X.Y.Z)

- 主版本号（X.0.0）：重大功能变更
  例如：1.0.0 → 2.0.0

- 次版本号（1.X.0）：新增功能
  例如：1.0.0 → 1.1.0

- 修订号（1.0.X）：Bug 修复
  例如：1.1.0 → 1.1.1
