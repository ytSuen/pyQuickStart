1. 下载 `pyQuickStart.exe`
2. 双击运行（首次运行需要管理员权限）
3. 开始使用快捷键管理功能

## 📝 使用说明

### 快捷键管理
- 点击"快捷键"输入框，按下你想设置的快捷键组合
- 快捷键会自动以大写加粗形式显示（如：CTRL+ALT+A）
- 输入目标路径（程序、网页、文件夹）
- 点击"添加快捷键"完成设置

### 防休眠功能
- 点击"开启防休眠"按钮启用
- 程序会智能防止系统休眠和锁屏
- 独立控制，不影响快捷键监听

### 联系发布者
- 点击界面底部的"发布者信息"链接
- 查看并复制发布者邮箱
